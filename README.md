## Emisor
Este micro:bit se encarga de recoger varios datos y enviaralos por radio a la otra micro:bit

## Edita este proyecto

Para editar este repositorio en MakeCode.

* abra [https://makecode.microbit.org/](https://makecode.microbit.org/)
* haga clic en **Import** luego haga clic en **Import URL**
* pegue **https://github.com/alelopez4623/meteo-microbit-emisor** y haga clic en importar
